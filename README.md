# Node server

## Usage

`node-server [options]`

#### Options

```
  -p, --port                        <PORT_NUMBER>
  -c, --cors                        <CORS_STRING>

  -lr, -lR, --logRequest            [<BOOLEAN>|<NUMBER>] (true)
  -lrh, -lRH, --logRequestHeaders   [<BOOLEAN>|<NUMBER>] (false)
  -lra, -lRA, --logRequestAddress   [<BOOLEAN>|<NUMBER>] (true)

  --rootLookForIndex                [<BOOLEAN>] (true)
```
